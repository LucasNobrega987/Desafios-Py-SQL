Análise de Interações entre Personagens
Desenvolvi um script Python chamado desafio1.py para analisar interações entre personagens a partir de dados armazenados em um arquivo CSV. Este script calcula o número total de interações para cada personagem em cada um dos três livros, além de uma soma geral dessas interações.

Configuração do Ambiente
Antes de começar a trabalhar no script, preparei meu ambiente de desenvolvimento da seguinte maneira:

Instalação do Python:

Certifiquei-me de instalar o Python em minha máquina, utilizando a versão 3.8. Você pode instalar o Python através do site oficial: Python.org.
Estrutura de Diretórios:

Organizei os diretórios de forma lógica, com uma pasta scripts para armazenar o arquivo desafio1.py e uma pasta data para o arquivo dataset.csv.
Execução do Script
Para rodar o script desafio1.py, segui estes passos:

Preparação dos Dados:

Coloquei o arquivo dataset.csv na pasta data. Este arquivo contém as informações necessárias para a análise.
Execução do Script:

Abri o terminal e naveguei até o diretório do projeto.
Executei o script usando o comando python scripts/desafio1.py.
O script processou os dados do CSV e gerou um arquivo de saída interacoes_resultados.csv no diretório especificado, contendo as interações calculadas.
Verificação dos Resultados
Após executar o script, verifiquei o arquivo interacoes_resultados.csv para confirmar se os resultados estavam corretos. Este arquivo contém colunas detalhadas para cada livro e o total de interações, ordenadas decrescentemente pelo total.

Documentação do Projeto
Documentei cada etapa do desenvolvimento e execução do script no arquivo README.md, proporcionando detalhes que permitem a qualquer pessoa entender e replicar o projeto. A documentação inclui informações desde a configuração do ambiente até a execução e verificação dos resultados.
